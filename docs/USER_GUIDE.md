## 1. เกี่ยวกับแอป

**Poatung (เป๋าตุง)** คือแอปบันทึกรายรับรายจ่ายส่วนตัว ที่ทำงาน **100% Offline** บนมือถือ ไม่ต้องใช้อินเทอร์เน็ต ไม่ต้องสมัครบัญชี

**คุณสมบัติหลัก:**
- บันทึกรายรับ/รายจ่ายพร้อมหมวดหมู่
- รองรับหลายกระเป๋าเงิน + โอนระหว่างกัน
- กราฟ Pie Chart, Bar Chart, Line Chart
- รายจ่ายประจำ (ค่าเช่า, Netflix ฯลฯ)
- เป้าหมายการออม
- ส่งออก CSV/HTML Report
- สำรอง/กู้คืนข้อมูล (JSON)
- ล็อคแอปด้วย PIN/Face ID/ลายนิ้วมือ
- แจ้งเตือนบันทึกรายจ่ายทุกวัน
- Dark Mode / Light Mode

## 2. การติดตั้ง

### สำหรับนักพัฒนา (Development)
```bash
# Clone project
cd D:\NoonStore-Poatung\expense-tracker

# ติดตั้ง dependencies
npm install

# รันแอป
npx expo start

# สแกน QR Code ด้วย Expo Go บนมือถือ
# หรือกด "a" เพื่อเปิดใน Android Emulator
```

### สำหรับ Build APK
```bash
# ติดตั้ง EAS CLI
npm install -g eas-cli

# Login เข้า Expo Account
eas login

# Build APK (preview)
eas build -p android --profile preview

# Build APK (production)
eas build -p android --profile production
```

## 3. หน้าจอหลัก (Dashboard)

เมื่อเปิดแอป จะเห็น **หน้า Dashboard** ประกอบด้วย:

| ส่วน | รายละเอียด |
|------|-----------|
| **BalanceCard** | แสดงยอดเงินรวมทั้งหมด, รายรับ/รายจ่ายเดือนนี้ |
| **กระเป๋าเงิน** | แถบเลื่อนแนวนอน แสดงกระเป๋าทั้งหมดพร้อมยอดเงิน |
| **ดูรายงาน** | ปุ่มไปหน้ารายงานและกราฟ |
| **รายการล่าสุด** | แสดง 5 รายการล่าสุด |
| **ปุ่ม + (FAB)** | กดเพื่อเพิ่มรายการใหม่เร็ว |

## 4. บันทึกรายรับรายจ่าย

**วิธีบันทึก:**
1. กดแท็บ **"บันทึก"** (ไอคอน + ตรงกลาง) หรือกดปุ่ม FAB ที่ Dashboard
2. เลือก **รายจ่าย** หรือ **รายรับ** (ปุ่ม toggle ด้านบน)
3. กดตัวเลขบน **Numpad** ใส่จำนวนเงิน
4. เลือก **หมวดหมู่** จาก Grid (เช่น อาหาร, เดินทาง)
5. เลือก **กระเป๋าเงิน** ที่ต้องการ
6. (เลือกได้) เปลี่ยนวันที่, เพิ่มโน้ต
7. กด **"บันทึก"**
8. แอปจะสั่นเบาๆ (Haptic) เพื่อยืนยัน

## 5. ประวัติรายการ

**วิธีดูประวัติ:**
1. กดแท็บ **"ประวัติ"**
2. ใช้ลูกศร ซ้าย/ขวา เลื่อนเดือน
3. ดูรายการแบ่งตามวัน พร้อมยอดรวมรายรับ/รายจ่ายแต่ละวัน

**ค้นหาและกรอง:**
- กดไอคอน 🔍 ค้นหาจากโน้ต
- กดปุ่ม **ทั้งหมด/รายจ่าย/รายรับ/โอน** เพื่อกรองประเภท
- กดเลือกหมวดหมู่เพื่อกรองเฉพาะหมวด

**ลบรายการ:**
- เลื่อนรายการไปทางซ้าย → กดปุ่ม **ลบ** สีแดง
- ยืนยันการลบ → ยอดเงินในกระเป๋าจะถูกปรับอัตโนมัติ

## 6. กระเป๋าเงิน

**เข้าถึง:** ตั้งค่า → กระเป๋าเงิน

**เพิ่มกระเป๋า:**
1. กดปุ่ม **"เพิ่มกระเป๋าเงิน"**
2. เลือก icon, ตั้งชื่อ, ใส่ยอดเริ่มต้น
3. กด **"บันทึก"**

**โอนเงินระหว่างกระเป๋า:**
1. กดปุ่ม **"โอนเงิน"**
2. เลือกกระเป๋าต้นทาง → ปลายทาง
3. ใส่จำนวนเงินผ่าน Numpad
4. กด **"โอนเงิน"**

## 7. รายงานและกราฟ

**เข้าถึง:** Dashboard → "ดูรายงานและกราฟ" หรือ ตั้งค่า → รายงานและกราฟ

| กราฟ | แสดง |
|------|------|
| **Summary Cards** (4 การ์ด) | รายรับ, รายจ่าย, คงเหลือ, ค่าเฉลี่ย/วัน + % เปรียบเทียบเดือนก่อน |
| **Pie Chart (รายจ่าย)** | วงกลมแยกตามหมวดหมู่รายจ่าย |
| **Pie Chart (รายรับ)** | วงกลมแยกตามหมวดหมู่รายรับ |
| **Bar Chart** | แท่งเปรียบเทียบรายรับ vs รายจ่ายรายสัปดาห์ |
| **Line Chart** | เส้นแนวโน้ม 6 เดือนย้อนหลัง |

## 8. รายจ่ายประจำ

**เข้าถึง:** ตั้งค่า → รายจ่ายประจำ

**ตั้งค่ารายจ่ายประจำ** (เช่น ค่าเช่า, Netflix):
1. กดปุ่ม **"เพิ่มรายจ่ายประจำ"**
2. เลือกประเภท (รายจ่าย/รายรับ)
3. ใส่จำนวนเงิน
4. เลือกความถี่ (ทุกวัน / ทุกสัปดาห์ / ทุกเดือน / ทุกปี)
5. เลือกหมวดหมู่, กระเป๋า, ใส่โน้ต
6. กด **"บันทึก"**

**ตรวจสอบรายการครบกำหนด:**
- กดปุ่ม **"ตรวจสอบรายการที่ครบกำหนด"** ด้านบน
- แอปจะสร้างรายการอัตโนมัติสำหรับรายการที่ถึงกำหนดแล้ว

## 9. เป้าหมายการออม

**เข้าถึง:** ตั้งค่า → เป้าหมายการออม

**สร้างเป้าหมาย:**
1. กดปุ่ม **"สร้างเป้าหมายใหม่"**
2. เลือก icon, ตั้งชื่อ (เช่น "เที่ยวญี่ปุ่น")
3. ใส่จำนวนเงินเป้าหมาย
4. (เลือกได้) ตั้ง deadline
5. กด **"บันทึก"**

**เพิ่มเงินเข้าเป้าหมาย:**
- กดปุ่ม **"เพิ่มเงิน"** ที่การ์ดเป้าหมาย
- ใส่จำนวนเงิน → กด **"เพิ่ม"**
- Progress Bar จะอัปเดตทันที
- เมื่อครบ 100% จะแสดงข้อความยินดี!

## 10. หมวดหมู่

**เข้าถึง:** ตั้งค่า → หมวดหมู่

**เพิ่มหมวดหมู่ใหม่:**
1. กดปุ่ม **"เพิ่มหมวดหมู่"**
2. เลือก Emoji (30 ตัวเลือก)
3. ตั้งชื่อ
4. เลือกสี (16 สี)
5. กด **"บันทึก"**

> หมวดหมู่เริ่มต้น (Default) ไม่สามารถลบได้ แต่แก้ไขได้

## 11. สำรองและกู้คืนข้อมูล

**เข้าถึง:** ตั้งค่า → สำรองและกู้คืนข้อมูล

### สำรองข้อมูล (Backup)
1. กดปุ่ม **"สร้าง Backup"**
2. แอปจะสร้างไฟล์ JSON บนเครื่อง
3. เลือกแชร์ไปยัง Google Drive / Line / Email / etc.

### กู้คืนข้อมูล (Restore)
1. กดปุ่ม **"เลือกไฟล์กู้คืน"**
2. เลือกไฟล์ JSON Backup
3. แอปจะแสดง preview (จำนวนรายการ, กระเป๋า, หมวดหมู่)
4. กด **"กู้คืน"** เพื่อยืนยัน
5. ⚠️ ข้อมูลปัจจุบันจะถูกลบทั้งหมดและแทนที่ด้วย Backup

## 12. ส่งออกรายงาน

**เข้าถึง:** ตั้งค่า → ส่งออกรายงาน

### Export CSV
- เลือกเดือน → กด **"ส่งออก CSV"**
- ไฟล์ CSV รองรับภาษาไทยใน Excel (BOM UTF-8)
- เปิดใน Excel, Google Sheets, Numbers ได้

### Export HTML Report
- เลือกเดือน → กด **"ส่งออก HTML"**
- รายงานสวยงามพร้อมสรุปรายรับ/รายจ่าย
- เปิดในเบราว์เซอร์แล้วพิมพ์เป็น PDF ได้

---

## 13. ล็อคแอป

**เข้าถึง:** ตั้งค่า → ล็อคแอป

### ตั้งค่า PIN
1. เปิด Switch **"ล็อคแอป"**
2. กรอก PIN 4-6 หลัก
3. ยืนยัน PIN อีกครั้ง
4. เรียบร้อย! แอปจะล็อคทุกครั้งที่เปิดหรือกลับจาก background

### Biometric (Face ID / ลายนิ้วมือ)
- ถ้าเครื่องรองรับ จะแสดงตัวเลือก Biometric อัตโนมัติ
- กดปุ่ม **"ทดสอบ"** เพื่อทดสอบ Biometric
- เมื่อเปิดแอป จะลอง Biometric ก่อน ถ้าไม่สำเร็จจึงใช้ PIN

### ปิดล็อค
- ปิด Switch "ล็อคแอป" → กรอก PIN เพื่อยืนยัน

---

## 14. การแจ้งเตือน

**เข้าถึง:** ตั้งค่า → การแจ้งเตือน

1. เปิด Switch **"เตือนบันทึกรายจ่าย"**
2. เลือกเวลาที่ต้องการ (07:00 - 22:00)
3. แอปจะแจ้งเตือนทุกวันตามเวลาที่ตั้ง
4. กดปุ่ม **"ส่งแจ้งเตือนทดสอบ"** เพื่อทดสอบ

| 3.14 | เปิดแจ้งเตือน | ตั้งค่า → การแจ้งเตือน → เปิด Switch | ⬜ |
| 3.15 | เลือกเวลา | การแจ้งเตือน → เลือกเวลาอื่น (เช่น 19:00) | ⬜ |
| 3.16 | ทดสอบแจ้งเตือน | การแจ้งเตือน → "ส่งแจ้งเตือนทดสอบ" → เห็น notification | ⬜ |
