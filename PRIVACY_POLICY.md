# นโยบายความเป็นส่วนตัว — Poatung (เป๋าตุง)

**วันที่มีผลบังคับใช้:** 30 มีนาคม 2569 (2026)
**พัฒนาโดย:** SEVENDOG DEV

---

## 1. ข้อมูลทั่วไป

แอปพลิเคชัน Poatung (เป๋าตุง) เป็นแอปบันทึกรายรับรายจ่ายส่วนตัว ที่ทำงานแบบ **Offline-First** บนอุปกรณ์มือถือของผู้ใช้ เราให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้เป็นอย่างยิ่ง

## 2. ข้อมูลที่เก็บรวบรวม

### ข้อมูลที่เก็บบนเครื่องเท่านั้น (Local Storage)
- รายการรายรับ/รายจ่าย (จำนวนเงิน, วันที่, หมวดหมู่, โน้ต)
- ข้อมูลกระเป๋าเงิน (ชื่อบัญชี, ยอดเงิน)
- หมวดหมู่ที่กำหนดเอง
- เป้าหมายการออม
- รายจ่ายประจำ (Recurring Rules)
- การตั้งค่าแอป (ธีม, เวลาแจ้งเตือน)

### ข้อมูลที่ **ไม่** เก็บรวบรวม
- ข้อมูลส่วนบุคคล (ชื่อ, อีเมล, เบอร์โทร)
- ข้อมูลตำแหน่งที่ตั้ง (GPS)
- ข้อมูลการใช้งานแอปอื่น
- ข้อมูลการท่องเว็บ
- ข้อมูลรูปภาพหรือไฟล์อื่นๆ
- ข้อมูลบัญชีธนาคารจริง

## 3. การจัดเก็บข้อมูล

- ข้อมูลทั้งหมดถูกเก็บใน **ฐานข้อมูล SQLite บนเครื่องของผู้ใช้เท่านั้น**
- ข้อมูลจะ **ไม่ถูกส่งไปยังเซิร์ฟเวอร์ภายนอก** ไม่ว่ากรณีใดๆ
- รหัส PIN ถูกเก็บอย่างปลอดภัยใน Secure Store ของระบบปฏิบัติการ

## 4. การสำรองข้อมูล (Backup)

- ผู้ใช้สามารถเลือกสำรองข้อมูลเป็นไฟล์ JSON ได้ด้วยตัวเอง
- ไฟล์ Backup จะถูกสร้างบนเครื่องและผู้ใช้เลือกแชร์เองตามต้องการ
- แอปไม่ส่ง Backup อัตโนมัติไปยังที่ใด

## 5. การแจ้งเตือน (Notifications)

- แอปใช้ Local Notifications เพื่อเตือนบันทึกรายจ่าย
- การแจ้งเตือนเป็น **แบบ Local เท่านั้น** ไม่มีการใช้ Push Server ภายนอก
- ผู้ใช้สามารถเปิด/ปิดการแจ้งเตือนได้ตลอดเวลาในหน้าตั้งค่า

## 6. การรักษาความปลอดภัย

- แอปรองรับการล็อคด้วย PIN, Face ID, และลายนิ้วมือ
- PIN ถูกเก็บใน Secure Store ของระบบปฏิบัติการ (Keychain บน iOS, Keystore บน Android)
- ข้อมูลไม่ถูกส่งผ่านเครือข่ายอินเทอร์เน็ต

## 7. สิทธิ์ที่แอปขอใช้ (Permissions)

| สิทธิ์ | เหตุผล |
|--------|--------|
| USE_BIOMETRIC / USE_FINGERPRINT | สำหรับการล็อคแอปด้วยลายนิ้วมือ/Face ID |
| Notifications | สำหรับเตือนบันทึกรายจ่ายประจำวัน |
| File Access (เลือกไฟล์) | สำหรับ Import ไฟล์ Backup เท่านั้น |

## 8. ข้อมูลของเด็ก

แอปนี้ไม่ได้ออกแบบมาสำหรับเด็กอายุต่ำกว่า 13 ปี และไม่เก็บข้อมูลส่วนบุคคลของเด็กโดยเจตนา

## 9. การเปลี่ยนแปลงนโยบาย
